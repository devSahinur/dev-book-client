## About & Features

**devBooks** is a e-commerce electronic website. It have an Admin Panel page. It have Its won database. You can post your product by uploading in database in the admin panel add product section. You also can delete any products from database by delete action of manage page. It has login authentication and orders details section.

### The website is built with :

### Front-end Technology : 
- **React, React-Router-Dom,**
- **React-Bootstrap**
- **CSS3**
- **Firebase Authentication**


### Back-end Technology : 
- **Node**
- **Express**
- **MongoDB**
- **dotenv**
